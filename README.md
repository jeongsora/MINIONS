<h4>미니언즈 소개 홈페이지 기획 및 제작</h4>
일루미네이션 스튜디오의 장편 애니메이션 '슈퍼배드' 에서 인기를 얻은 미니언들을 주인공으로 한 스핀오프(spin-off) 작품입니다.<br>

제가 좋아하는 애니메이션 '미니언즈'를 소개하는 사이트를 제작하였습니다.<br>
원페이지로 제작하였고, 각각의 캐릭터 인적사항과 예고편이 들어있습니다.<br><br>

- 제작기간 : 2일<br>
- 홈페이지 유형 : 원페이지<br>
- 사용언어 : HTML, CSS, JQuery, Javascript<br>
